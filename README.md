# Kinevo

Kinevo is a premium UX/UI fitness application focused on providing a clean, accessible, and user-centered experience.

This repository contains **UI only** code developed for an academic UX/UI project.
No backend, no real AI integration, and no sensitive data are included.

## Project Scope
- UX/UI focused
- Frontend only
- No authentication
- No external services
- Safe for public GitHub repositories

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
